# **App Name**: Dream Weaver

## Core Features:

- Dream Input: Text input area for users to describe their dreams. Ripples or shimmers as users type.
- Dream NFT Forge: Transforms dream descriptions into abstract art NFTs. Uses generative AI tool.
- NFT Preview: Dream NFTs displayed as liquid, evolving visuals.
- Global Dream Index Map: A constellation of dreams, glowing nodes connect shared patterns across the world. Allows users to select different levels of global emotional experiences.
- Dream Saving: Allows users to save their dream and dream NFT, linking to their user id in Firestore.

## Style Guidelines:

- Primary color: Moonlit violet (#4C51A8), embodying the mystical and introspective nature of dreams.
- Background color: Soft blue (#D1D5DB), creating a hazy, ethereal backdrop.
- Accent color: Cosmic gold (#B45309), adding a touch of the surreal and cosmic.
- Font: 'Belleza' for headlines (sans-serif), and 'Alegreya' for body (serif).
- Logo suggestion: a merging of a dreamcatcher and blockchain node. Minimalist, flowing icons representing dream states as visual auras.
- Soft surreal gradients, hazy edges, and floating elements throughout the interface. Motion design inspired by REM sleep eye movement and breathing rhythms.
- Slow-motion particle trails and whisper-like animations that are subtle, hinting at deeper experiences.
